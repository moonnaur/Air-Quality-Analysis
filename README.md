# Air-Quality-Analysis

This project focused on analyzing air quality and greenhouse gas emissions across the United States using data science techniques, web scraping, APIs, and data visualization. The analysis combined air quality index (AQI) data, greenhouse gas emissions, energy consumption, and population statistics to explore how population density and energy usage impact environmental conditions. Data was collected from multiple online sources using APIs, Selenium web scraping, and reverse geocoding to transform geographic coordinates into usable city and state information.

After cleaning and processing the datasets with Python, the project produced several visualizations including AQI maps, greenhouse gas emission breakdowns by sector, time series analyses, scatter plots, and state-level comparisons. The findings showed that areas with higher population density generally had higher energy consumption and greenhouse gas emissions, with the energy sector contributing the largest share of emissions nationwide. The project demonstrated skills in data acquisition, data cleaning, API integration, web scraping, exploratory data analysis, and visualization using Python libraries to communicate environmental trends and support sustainability-focused insights.
